# 快速开始指南

## 🚀 一键启动

### 方法一：使用快速启动脚本
```bash
cd game
./install.sh    # 首次运行安装依赖
./start.sh      # 启动游戏
```

### 方法二：手动启动
```bash
# 终端1 - 启动后端服务器
cd game/server
npm install  # 首次运行
npm start

# 终端2 - 启动前端开发服务器
cd game/client
npm install  # 首次运行
npm run dev
```

## 🎮 开始游戏

1. **访问前端**：打开浏览器访问 http://localhost:5173
2. **创建房间**：输入你的名字，点击"创建房间"
3. **等待玩家**：分享房间ID给朋友，或让朋友加入你的房间
4. **准备游戏**：所有玩家点击"准备"按钮
5. **开始游戏**：所有人准备好后自动开始

## 📋 游戏流程

1. **抽取板块**：系统随机抽出一片六角板块
2. **放置板块**：点击图板上的空位放置当前板块
3. **等待回合**：所有玩家放置完毕后抽取下一片
4. **计算分数**：放置完19片后游戏结束，显示分数排名

## 🎯 得分规则

- 每条完整得分线 = **数字 × 片数**
- 三个方向：垂直、左上右下、右上左下
- 线上所有数字必须相同才能得分
- 满分：**307分**

## 🛠️ 开发

### 项目结构
- `client/` - React前端应用
- `server/` - Node.js后端服务器
- `shared/` - 共享的游戏逻辑

### 测试游戏逻辑
```bash
cd game
node test-game.js
```

## ❓ 常见问题

**Q: 服务器启动失败？**
A: 确保端口3001未被占用

**Q: 前端无法连接后端？**
A: 检查后端服务器是否运行在 http://localhost:3001

**Q: Socket连接失败？**
A: 确保防火墙允许WebSocket连接

## 📝 注意事项

- 至少需要2名玩家才能开始游戏
- 每局游戏最多6名玩家
- 放好的板块无法移动
- 所有玩家使用相同的板块进行游戏
